# Aditya Pandey — Portfolio

Personal portfolio site: fintech platform work, Open Banking payments, and AI tooling, presented as case studies.

Plain HTML/CSS/JS — no build step. Hosted on GitHub Pages.

## Develop

Open `index.html` in a browser. That's it.

## Publish

```powershell
./publish.ps1
```

Logs you into GitHub (personal account) if needed, creates `<username>.github.io`, and pushes. Subsequent runs just push updates.
